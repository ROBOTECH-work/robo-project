HTML Adventures

🌐 HTML (HyperText Markup Language) - Building Blocks of Web Pages

HTML is the language used to create and structure websites. It's like the skeleton of a web page, telling the computer what each part of the page is.

Basic Structure of an HTML Document:
<!DOCTYPE html>
<html>
  <head>
    <title>My First Webpage</title>
  </head>
  <body>
    <h1>Hello, world!</h1>
    <p>This is a paragraph.</p>
  </body>
</html>
Common HTML Tags
Tag Purpose
<h1> to <h6> Headings (from largest to smallest)
<p> Paragraph
<a href=""> Link
<img src=""> Image
<ul> / <li> Unordered List and List Items
<ol> Ordered List
<div> Container for content
<br> Line break
Important Concepts

Attributes give extra info to tags:

<img src="cat.jpg" alt="A cute cat" width="200" />

Nesting: HTML tags can be placed inside others.

CSS Magic

🎨 CSS (Cascading Style Sheets) - Styling Web Pages

CSS is used to make HTML content look beautiful by changing colors, sizes, fonts, and more.

CSS Syntax
selector {
  property: value;
}

Example:

h1 {
  color: red;
  font-size: 40px;
}
Ways to Use CSS
  • Inline (inside an HTML tag)
    <p style="color:blue;">This is blue text.</p>
  • Internal (within <style> tag)
    <style>
p { color: green; }
</style>
  • External (linked CSS file)
    <link rel="stylesheet" href="styles.css">
Common Properties
Property What It Does
color Changes text color
background-color Changes background color
font-size Sets text size
margin Adds space outside elements
padding Adds space inside elements
border Adds a border

Robotics Fun

🧐 Robotics - Learning About Smart Machines

A robot is a machine that can be programmed to do tasks on its own. Robots can sense, think, and act.

Main Parts of a Robot
  • Controller (Brain) – A microcontroller like Arduino or Raspberry Pi
  • Sensors – Eyes and ears (e.g. camera, light sensor)
  • Actuators – Arms and wheels (motors, servos)
  • Power Supply – Batteries or plugs
What Can Robots Do?
  • Follow lines
  • Avoid obstacles
  • Clean houses (Roomba)
  • Build cars in factories
Fun Projects
  • Line-following robot
  • Drawing robot
  • Light-sensitive robot

Scratch Coding

🐝 Scratch - Drag and Drop Coding

Scratch is a visual programming language designed for kids. It uses blocks to build games, animations, and stories.

Main Categories
Category What it Does
Motion Move, turn, glide
Looks Say, show, change color
Sound Play sounds and music
Events Start when flag clicked
Control Repeat, wait, if...then
Sensing Detect touching, loudness, etc.
Sample Project
  • When green flag clicked
  • Move 10 steps
  • Say "Hello!"
  • Play a meow sound
Scratch Website

Go to https://scratch.mit.edu to create your own projects.

Python Turtle

📚 Python Turtle - Drawing with Code

Turtle is a built-in Python module that helps you draw shapes and pictures using code.

Example
import turtle

t = turtle.Turtle()
t.forward(100)
t.right(90)
t.forward(100)
Common Commands
Command Action
forward(100) Move forward
backward(100) Move backward
right(90) Turn right
left(90) Turn left
color("red") Change pen color
begin_fill() & end_fill() Fill shapes

Install Python

🤖 Installing Python (Windows)

Steps to Install Python
  1. Go to the website: https://www.python.org
  2. Click on Downloads and select Python for Windows
  3. Run the downloaded file
  4. Make sure you check the box that says Add Python to PATH
  5. Click Install Now
  6. After installing, open Command Prompt and type: 
    python --version
  7. You should see something like: 
    Python 3.12.1

Now Python is ready to use!

File Extensions

🗋 File Extensions - What They Mean

A file extension is the last part of a file name, like .txt, .jpg, .html. It tells your computer what kind of file it is.

Common Extensions
Extension Type of File
.txt Plain text file
.jpg / .png Picture files
.html Webpage file
.css Styling file
.py Python file
.sb3 Scratch project file
.docx Word document
.mp3 Audio file
.mp4 Video file
Important Note

Always be careful not to change the extension unless you know what you're doing!
