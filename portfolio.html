Featured Projects

Drone Project

This project introduces kids to the exciting world of drone technology. Together, we build functional drones from scratch using lightweight frames, motors, propellers, and flight controllers. The children learn basic aerodynamics, electronics, and coding principles by assembling and programming the drones themselves.

Through this hands-on experience, they gain practical STEM skills while exploring creativity and teamwork. The project not only inspires curiosity about robotics and engineering but also helps them understand real-world applications of drones in areas like agriculture, photography, and disaster response.

Educational Robot Kit

In this project, students built a simple Arduino-powered robot using basic components like motors, wheels, an ultrasonic sensor, and a motor driver. The robot was programmed to move, avoid obstacles, and follow basic commands.

This hands-on experience helped students understand the fundamentals of electronics, programming, and robotics. They learned how sensors interact with code and how robots make decisions based on real-time input. It's a fun and practical way to introduce young learners to the world of embedded systems and automation.

Smart Garden

The Smart Garden project uses Arduino technology to automate and monitor plant care. Students built a system that uses soil moisture sensors, temperature sensors, and a water pump to detect when plants need watering and irrigate them automatically.

Through this project, learners explored environmental sensing, data collection, and automation. It’s an excellent way to introduce concepts like IoT (Internet of Things), sustainability, and practical problem-solving in agriculture using simple Arduino components.
